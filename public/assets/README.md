## Tips untuk iPhone Safari:
- Foto: gunakan format JPG/WEBP, ukuran < 2MB per foto
- Video: gunakan format MP4 (H.264), ukuran < 30MB
- Video codec: H.264 + AAC audio agar kompatibel dengan Safari iOS
